# 花间集 HUAJIANJI

> 中国城市花卉打卡互动地图 -- 一花一城，一步一景

## 简介

「花间集」是一个以中国传统水墨风格设计的花卉打卡互动地图。探索中国各城市的市花，了解它们的花语、花期与赏花圣地。

## 技术栈

- **后端**: Node.js + Express
- **前端**: HTML + CSS + JavaScript（原生，无框架依赖）
- **数据**: JSON 数据 + REST API

## 功能特性

- SVG 中国地图 + 水墨风格背景
- 18 个城市市花标记（emoji 花朵图标）
- 花卉详情卡片（城市、花卉名、花语、赏花地、花期）
- 底部可展开花卉列表面板
- 春夏秋冬季节筛选
- 地图拖拽平移、滚轮缩放
- 打卡统计展示
- 花瓣粒子飘落动画
- 中国传统书法字体 + 水墨渐变装饰

## 快速启动

```bash
# 安装依赖
npm install

# 启动服务
npm start

# 开发模式
npm run dev
```

启动后访问 http://localhost:3000

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/flowers` | 获取所有花卉数据 |
| GET | `/api/flowers/:id` | 获取单个花卉详情 |
| GET | `/api/flowers/season/:season` | 按季节筛选花卉 |
| GET | `/api/cities` | 获取所有城市坐标 |
| GET | `/api/stats` | 获取统计数据 |
